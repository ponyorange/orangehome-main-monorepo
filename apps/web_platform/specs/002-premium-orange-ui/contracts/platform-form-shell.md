# UI Contract: 平台表单壳与暖橙主界面

**Feature**: `002-premium-orange-ui`  
**Date**: 2025-03-27

## 适用范围

- **创建项目** Modal（Projects）
- **新建页面** Modal（Project Detail）
- **登录后主壳**：Header、MainLayout 内容区、项目列表与详情页主要 Card/Table 容器（分阶段时可标注「Phase 1 / Phase 2」）

## 创建类 Modal（两者必须一致）

| # | 检查项 | 期望 | 创建项目 | 新建页面 |
|---|--------|------|----------|----------|
| 1 | Modal 标题 | 字重、字号层级高于表单内标签；与 Semi 默认相比更克制（避免轻飘） | ✓ | ✓ |
| 2 | 表单布局 | 垂直布局；字段纵向间距统一 | ✓ | ✓ |
| 3 | 标签与控件 | 标签与输入区对齐方式一致；占位符对比度可读 | ✓ | ✓ |
| 4 | 主按钮 | 暖橙实心（或与平台 `--oh-primary` 一致），非冷蓝 | ✓ | ✓ |
| 5 | 次要按钮 | 取消为弱样式（tertiary/borderless），与主按钮层级分明 | ✓ | ✓ |
| 6 | 底部操作区 | 同一对齐方式（建议右对齐）、按钮顺序一致：取消 → 提交 | ✓ | ✓ |
| 7 | 错误态 | 校验信息与字段距离固定，不遮挡相邻字段 | ✓ | ✓ |
| 8 | 圆角与边线 | Modal 内容区与表单控件圆角与平台 Card 规则同一套 token | ✓ | ✓ |

**走查说明（2026-03-29）**：两 Modal 共用 `className="oh-form-modal"` 与 `src/styles/form-modal.scss`；提交按钮 `type="primary"` + `theme="solid"`；实现以 `quickstart.md` A/B 节人工复核为准。

## 主界面（P3）

- 顶栏与内容区背景有 **轻微层次**（非刺眼渐变）。
- 全局 **链接色** 不与暖橙主调冲突（不用默认冷紫/冷蓝作为主链色）。
- 表格表头、卡片标题 **字重/字号** 层级可读。

## 回归

- 创建项目 / 创建页面 **提交成功与失败** 路径与改版前一致（字段与 API 不变）。
