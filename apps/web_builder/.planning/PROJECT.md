# PROJECT.md

## 项目概述

**Orange Editor**（@orangehome/web_builder）是一个基于 React + TypeScript 构建的可视化页面编辑器，采用模块化、插件化的架构设计。主要用于构建和编辑活动页面，支持组件拖拽、属性配置、实时预览等功能。

### 基本信息

| 属性 | 值 |
|-----|-----|
| **项目名称** | Orange Editor |
| **包名** | @orangehome/web_builder |
| **构建工具** | Vite |
| **目标环境** | ES2022 |
| **作者** | orangehome |

### 核心特性

- 🎨 **可视化编辑** - 拖拽组件、实时预览
- 🧩 **插件化架构** - 基于 Inversify 的 IoC 容器，支持扩展
- 📐 **Schema 驱动** - JSON Schema 定义页面结构
- 🎯 **原子组件** - 50+ 配置表单组件
- 🚀 **性能优化** - 首屏渲染追踪、性能监控
- 🌐 **远程组件** - 动态加载远程组件

## 设计原则

1. **IoC 优先** - 所有功能通过 Inversify 容器管理依赖
2. **扩展驱动** - 新功能以 Extension 形式添加，不修改核心
3. **Schema 中心** - 页面结构由 Schema JSON 驱动，编辑器只是操作 Schema 的工具
4. **类型安全** - 完整的 TypeScript 类型覆盖
5. **性能优先** - 首屏渲染、操作响应都有性能监控
