# Quickstart: 006-canvas-750x1334-rulers

## 环境

```bash
cd apps/web_builder
rushx dev
```

## 验收步骤

1. 打开编辑器，看顶栏（或等价位置）画布尺寸文案 → **750×1334**（或与合同常量一致）。  
2. 水平标尺：滚动到画布右缘，末端刻度为 **750**；垂直标尺末端为 **1334**。  
3. 画布白底区域逻辑大小：用浏览器开发者工具测量缩放后的像素宽高 ≈ `750 * zoom` × `1334 * zoom`。  
4. 切换 **2 个不同缩放比例**，标尺与画布边缘仍对齐，无「标尺范围仍是旧尺寸」的脱节。  
5. 拖拽组件、对齐线、右键菜单等基本操作可用。  
6. （可选）打开含旧坐标的 JSON 页面：确认无白屏；是否提示幅面变化由实现决定。

## 回归检索

在 `apps/web_builder/src` 内搜索 `375`、`667`（排除颜色值 `#667085` 等），确认无**指代画布宽高**的残留。
