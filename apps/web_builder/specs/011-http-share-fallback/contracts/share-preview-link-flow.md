# Contract: 运行时预览链接「分享」流程

## 入口

- **UI**：编辑器工具栏「分享」按钮（与现网一致）。
- **输入**：当前上下文的 `pageId`（与 `buildRuntimePreviewUrl` 一致）。

## 输出语义

| 结果 | 用户可见行为 |
|------|----------------|
| **无 URL** | 与现网一致：提示无法生成链接（配置/pageId 原因），**不**进入复制/Modal。 |
| **复制成功** | 成功 Toast；**不**展示完整链接 Modal。 |
| **复制失败（URL 已生成）** | 打开 Modal：完整 URL 可读；提供「新标签页打开」；可关闭 Modal。 |

## 复制子流程（实现契约）

1. **尝试 A**：异步剪贴板 API（若可用且未抛错 → 成功）。  
2. **尝试 B**：`execCommand('copy')` + 临时输入控件（成功 → 成功）。  
3. **失败**：进入上表「复制失败」Modal 路径。

## 新开标签页

- **MUST** 使用 `window.open(url, '_blank', 'noopener,noreferrer')`（或等价安全组合）。
- 若返回值表明被拦截，MUST 向用户给出可理解提示（见 spec Edge Cases）。
