# Data Model: 007 编辑区半屏展示

本特性 **不引入持久化实体**，仅在编辑器内区分三类 **概念量**（供实现与测试对齐 spec **Key Entities**）。

## 1. 逻辑画布（设计幅面）

| 字段 | 说明 |
|------|------|
| 逻辑宽 | 固定 **750**（`EDITOR_CANVAS_WIDTH`） |
| 逻辑高 | 固定 **1334**（`EDITOR_CANVAS_HEIGHT`） |
| 权威来源 | `ISchema` 及属性面板中的位置/尺寸；与 006 合同一致 |

**规则**：不因本特性把全宽组件的语义改为 375 或其它值。

## 2. 用户缩放（设计空间）

| 字段 | 说明 |
|------|------|
| `zoom` | `useZoom` 状态，默认 `1.0`，范围与步进保持现有实现 |
| 用户可见读数 | `formatZoomPercent(zoom)`，**不**应仅因半屏展示而显示为 50% |

**规则**：`zoom` 表示相对 **逻辑幅面** 的缩放意图，而非「再乘一次展示比例后的复合标签」。

## 3. 视口展示比例（仅中心编辑区）

| 字段 | 说明 |
|------|------|
| `EDITOR_VIEWPORT_SCALE`（命名以代码为准） | 常量，默认 **0.5**，表示相对当前实现的 **线性** 视觉缩放 |
| 布局占位宽（像素） | `EDITOR_CANVAS_WIDTH * zoom * EDITOR_VIEWPORT_SCALE` |
| 布局占位高（像素） | `EDITOR_CANVAS_HEIGHT * zoom * EDITOR_VIEWPORT_SCALE` |
| 内层排版宽（像素） | `EDITOR_CANVAS_WIDTH * zoom`（与改造前一致，用于 schema 子树布局） |

**规则**：展示比例为 **编辑器视口策略**，**不写回** schema；默认 **不提供** 用户开关（与 spec 范围一致；若产品后续需要，再升为配置项）。

## 关系小结

```text
逻辑幅面 (750×1334)  ——不变——>  schema / 属性
        × zoom        ——不变——>  useZoom / 顶栏百分比
        × VIEWPORT_SCALE ——新增——> 中心区滚动占位、标尺像素长度、（必要时）指针换算
```

## 校验规则（来自 FR）

- **FR-004**：指针操作后的 schema 坐标与 **逻辑幅面** 一致。
- **FR-005**：标尺末端刻度语义仍对应 **750 / 1334**，而非 375。
