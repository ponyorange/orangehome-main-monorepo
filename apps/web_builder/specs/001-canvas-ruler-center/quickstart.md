# Quickstart: 验收画布与横向标尺水平居中

## 前置

- 在 `apps/web_builder` 执行 `rushx dev`（或 monorepo 中等价命令）打开编辑器。
- 使用桌面浏览器；开发者工具可开可关。

## P1 — 窗口宽度

1. 左、右侧栏保持常规展开。  
2. 将浏览器窗口从宽逐步拖窄，再拉宽，重复至少 5 次不同宽度。  
3. **期望**：画布手机框与上方横向标尺作为整体，在中间画布区域内视觉水平居中，无长期偏左/偏右。  
4. 改变画布缩放（+ / −），重复步骤 2–3。  
5. **期望**：缩放后仍居中。

## P2 — 侧栏

1. 固定浏览器窗口宽度。  
2. 切换左侧面板折叠/展开（或任何会改变中间列可用宽度的操作），至少 3 种状态。  
3. **期望**：每种状态稳定后，画布与横向标尺仍在中间列内水平居中。

## 回归

- 画布内拖拽、选中、对齐线、拖入组件、上下文菜单仍正常。  
- 横向标尺刻度与画布内容水平对齐无异常错位。

## 质量命令

```bash
cd apps/web_builder
rushx type-check
rushx lint
```
