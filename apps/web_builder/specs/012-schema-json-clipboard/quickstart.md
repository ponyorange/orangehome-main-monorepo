# Quickstart: 012 Schema JSON 编辑器剪贴板快捷键

## 前置

- 打开任意页面进入编辑器。
- 在右侧属性面板切换到「信息」页，点击「编辑 Schema」打开弹窗（见 `PropertiesPanel.tsx`）。

## 1. 编辑器内复制/粘贴（US1）

1. 在 Schema JSON 编辑器里选中一段文本。  
2. 按 Ctrl/⌘+C。  
3. 将光标移动到另一处，按 Ctrl/⌘+V。  
4. **预期**：文本被粘贴；画布不发生组件复制/粘贴导致的结构变化。

## 2. 外部 → 编辑器粘贴（US2）

1. 在外部应用（记事本/IM/浏览器其它输入框）复制一段**合法且结构有效**的 JSON。  
2. 回到 Schema JSON 编辑器粘贴。  
3. **预期**：文本粘贴成功；若 JSON 合法且结构有效，画布同步更新；若非法/结构无效，提示错误且画布保持上一次有效状态。

## 3. 编辑器 → 外部（US3）

1. 在 Schema JSON 编辑器选中一段 JSON。  
2. Ctrl/⌘+C 后切换到外部应用粘贴。  
3. **预期**：外部应用得到与选区一致的文本。

## 4. 回归：画布快捷键不受影响

1. 关闭 Schema 编辑弹窗，确保焦点回到画布区域。  
2. 选中画布组件，按 Ctrl/⌘+C、Ctrl/⌘+V。  
3. **预期**：仍按原有“组件复制/粘贴”行为工作。
