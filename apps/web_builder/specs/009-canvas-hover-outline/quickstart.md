# Quickstart: 009 画布悬停虚线框

## 前置

- 本地启动 web_builder 编辑器，进入带多个组件（含嵌套）的页面。  
- 确认画布使用可选择渲染路径（`CenterCanvas` + `SelectableSchemaRenderer`）。

## 验证 SC-001 / US1

1. 在画布区内缓慢移动鼠标，依次经过 **2 个以上**组件主体。  
2. 预期：指针停在每个组件上时，约 **200ms 内**出现**虚线框**，且与组件边缘对齐。  
3. 将指针移到根容器**留白**（无子组件覆盖区域）：预期**不出现**针对根的错误子级虚线（与规格根策略一致）。

## 验证 SC-002 / FR-006

1. **选中**某组件，观察选中框。  
2. 将指针移到**同一组件**上（若实现为「选中不显示 hover」则跳过对比，改为对比**未选中**节点）：悬停虚线与选中框应 **同源测量**，无明显漂移。  
3. 变更画布**缩放**与**滚动**后重复悬停，框仍对齐。

## 验证 US2 / FR-004

1. 选中组件 A。  
2. 指针移到未选中的组件 B。  
3. 预期：A 仍为选中态；B 显示悬停虚线（若 B 未选中）；**选中集合未变**。  
4. 指针移出所有组件：悬停虚线消失。

## 验证 SC-003

1. 在画布上对有 **输入框类**物料（若有）点击、输入。  
2. 预期：可正常聚焦与输入；悬停层不阻挡点击。
