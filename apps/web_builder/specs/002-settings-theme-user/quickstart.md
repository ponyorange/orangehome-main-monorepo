# Quickstart: 设置主题与顶栏用户展示

## 前置

- `apps/web_builder` 执行 `rushx dev`，使用已可登录的环境。  
- 准备两个账号（可选）：一者有头像+昵称；一者无头像仅有邮箱。

## P1 — 设置与主题

1. 登录并进入编辑器主界面。  
2. 在顶栏**左侧**找到**设置**入口，点击。  
3. 确认下拉出现且含 **「切换主题」**。  
4. 点击该项，确认出现主题切换 UI（色点/列表与改前能力一致），切换主题后顶栏与全局变量同步。  
5. 点击菜单外或 Esc，确认菜单关闭；再次打开仍可用。

## P2 — 右上角用户

1. 使用有头像账号：确认 **头像 + 昵称** 在**顶栏右侧**。  
2. 使用无头像账号：确认 **默认头像**（渐变/首字）+ 昵称或邮箱展示。  
3. 将浏览器宽度缩至约 1280px：确认名称省略号、不与设置按钮重叠。

## 回归

- Logo、中间 Toolbar、保存类 Actions 仍可点击。  
- 刷新页面后主题与用户信息仍正确。

## 命令

```bash
cd apps/web_builder
rushx type-check
rushx lint
```
