# Quickstart: 010 已保存状态与预览前保存

## 前置

- 登录、打开带 `pageId` 的编辑器页；网络可访问保存接口。

## 1. 初始已对齐

1. 刷新进入页面，等待加载完成。  
2. **不**做任何编辑，点「预览」。  
3. **预期**：无「先保存」长等待（无脏时不强制保存）；预览正常打开。

## 2. 编辑后预览触发保存

1. 拖动或改属性使 schema 变化。  
2. 点「预览」。  
3. **预期**：预览按钮（或等价 UI）出现 **loading**；保存成功后预览打开；运行时内容与编辑区一致。

## 3. 保存失败不打开预览

1. 断网或 Mock 保存失败。  
2. 有未保存编辑时点「预览」。  
3. **预期**：错误提示；**不**进入预览；仍为脏。

## 4. 手动保存后预览

1. 编辑后点「保存」成功。  
2. 点「预览」。  
3. **预期**：不应再触发完整保存链（或仅极短无感）；预览打开。

## 5. 导入 JSON

1. 导入新 schema。  
2. **预期**：视为未对齐；预览应先保存（若允许保存）。
