# Quickstart: 005-context-menu-follow-pointer

## 环境

无需新增环境变量。本地运行：

```bash
cd apps/web_builder
rushx dev
```

## 验收步骤（对齐 spec SC-001 / SC-002）

1. 打开带画布的编辑页，在画布**中央**某节点上 **右键** → 菜单应出现在指针附近（默认左上角贴近点击点）。  
2. 将窗口调小，在视口**右下角**附近右键 → 菜单应**完整可见**，主要项可点（允许相对指针向左/向上偏移）。  
3. 在视口**左上、右上、左下**各试一次 → 菜单均不应大面积裁出视口外。  
4. **滚动**中间画布区域后，在仍可见的节点上右键 → 菜单仍相对**浏览器视口**正确放置。  
5. 菜单打开后**移动鼠标**（不点选）→ 菜单**不**跟随指针漂移；点击空白关闭后，在另一位置再右键 → 按**新位置**锚定。
